

🧠 **Simple Quiz App**

An interactive Quiz Application built using HTML, CSS, and JavaScript.
This project dynamically displays questions, validates answers, and tracks the user's score.

---

🚀 **Live Demo**

🌍 Try the app here:  👉 https://java-script-quiz-app-ashen.vercel.app/



---
📌 **Features**

📋 Multiple choice quiz questions

✅ Instant answer validation

🎯 Score tracking system

➡️ Next question navigation

🔄 Restart quiz option

🎨 Clean and responsive UI

⚡ Built using pure JavaScript (no frameworks)

---

🛠️ **Technologies Used**

- HTML5 – Structure of the application

- CSS3 – Styling and layout

- JavaScript (ES6) – Logic and interactivity

⚙️ **How It Works**

- The quiz starts with the first question.

- Users select an answer from the options.

- The app highlights the correct and incorrect answers.

- Users move to the next question using the Next button.

- At the end, the app displays the final score.

- Users can restart the quiz using Play Again.
